# ChatApp

Fullstack app with realtime notification using socket.io

# MERN Stack 
-Mongo DB
-Express JS
-Reactjs 
-Nodejs

# Development
Client: React JS

Server: Node JS, Express JS

Database: Mongo DB

# Run Localy
Go to the project directory
- cd Chat-App

Install dependencies
  npm install
  

 -  cd frontend/
 -   npm install


 Start the server
    - npm run start

 Start the Client
    //open now terminal
    - cd frontend
     - npm start
